# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind.
- Assume good faith.
- Call in before you call out.
- Harassment, discrimination, and bad-faith participation are not tolerated.

## Scope

Applies in all project spaces: GitHub issues/PRs, Discord, any event under the Forage for All name, and public spaces where you represent the project.

## Enforcement

Report abuse to **conduct@forageforall.org**. Reports go to a small, rotating volunteer council — not to the person you're reporting.

Consequences range from a private warning to a permanent ban, depending on severity and pattern.

## Foraging-specific

- Respect that many traditional foraging practices are tied to Indigenous and immigrant communities. Don't appropriate; cite sources and credit teachers.
- Don't gatekeep. Someone new to foraging asking "is this edible?" is welcome.
- Don't dox locations for cultural or religious foraging practices without consent.
