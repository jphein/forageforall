## Translator guide

1. Copy `i18n/en.json` to `i18n/<your-locale>.json`
2. Translate values (keep keys in English)
3. Preserve interpolation tokens like `{{count}}` exactly
4. For species common names, use the **most widely recognized regional name**, not a literal translation
5. Test on a device set to your locale
6. Open a PR — add yourself to this file

## Style for translations

Match the English voice (see [`STYLEGUIDE.md`](./STYLEGUIDE.md)) — plainspoken, specific, no hype.

When your language has formal/informal registers, use informal. Foraging is neighborly.
