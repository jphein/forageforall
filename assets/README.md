# Asset placeholders

Drop your actual files here before building:

- `icon.png` — 1024×1024 app icon (no transparency)
- `splash.png` — 1284×2778 (or square, resizeMode contain) splash
- `adaptive-icon.png` — 1024×1024 Android adaptive foreground (safe zone ~68%)
- `favicon.png` — 48×48 (only needed if you ship web)

Until you add them, Expo will use defaults — the app still runs.
